# 🛍 Vara's Fashion Mart – Full Stack E-Commerce Website

## 📌 Project Overview

Vara's Fashion Mart is a full-stack e-commerce website developed as an internship project.

Users can:

* Register and Login
* Browse Products
* Search Products
* View Product Details
* Add Products to Cart
* Increase / Decrease Quantity
* Place Orders
* View Order History
* Cancel Orders

The website provides a modern shopping experience with responsive UI and backend integration.
